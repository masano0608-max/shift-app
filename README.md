# 藤澤さんシフト管理アプリ

Excelファイル「藤澤さんシフト.xlsx」を元にしたシフト管理アプリです。

## 機能

- **新規作成**: 年月・社員情報・集計・日別勤怠を入力
- **Excelから読み込み**: 既存のExcelシフト表をインポート
- **保存**: 入力データを履歴として保存（ブラウザのlocalStorageに保存）
- **履歴一覧**: 保存した履歴の一覧表示
- **編集**: 保存した履歴を編集（編集日時が記録されます）
- **PDFダウンロード**: シフト表をPDFで出力

## 起動方法

```bash
cd shift-app
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## 本番ビルド

```bash
npm run build
```

`dist` フォルダにビルド済みファイルが出力されます。静的ホスティング（Netlify、Vercel等）にデプロイ可能です。

## データの保存について

- データはブラウザのlocalStorageに保存されます
- 同じブラウザ・同じドメインでアクセスする限りデータは保持されます
- ブラウザのキャッシュ/データを削除すると履歴は消えます
- 重要なデータは定期的にPDFでダウンロードしてバックアップすることをお勧めします
